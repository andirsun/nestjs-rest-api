## Description

- Architecture Used in this project : Clean Architecture
- Type of Architecture : Octagonal Architecture
- Folder Patters : src -> Apps -> Logic inside every folder

## Installation
 
 - Clone the repo 
```bash
$ git clone https://github.com/Timugo/rest-api-backend.git
```
- Tnstall node modules
```bash
$ npm install
```

## Running the Backend

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
